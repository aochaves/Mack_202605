# Projeto CI/CD com GitHub Actions

Este projeto demonstra um pipeline completo de Integração Contínua (CI) 
e Deploy Contínuo (CD) usando GitHub Actions.

## 📁 Estrutura
- `site/` - Arquivos estáticos (HTML, CSS, JS)
- `.github/workflows/` - Configurações dos workflows

## 🚀 Workflows
- `ci.yml` - Validação, testes e análise
- `cd.yml` - Deploy no GitHub Pages
